# Layo: Bridging Borders, Simplifying Logistics

Layo is a premium **logistics aggregator platform** designed to solve the significant price disparity for essential goods (specifically apparel and luxury items) between India and Canada.

## 🚀 Phase 1: The Intelligent Dashboard

We have successfully implemented the core dashboard with three distinct shipment options:

1.  **Selection Flow**: A granular item selection system with real-time weight and cost estimation based on our master data of over 100+ items.
2.  **WhatsApp Concierge**: A white-glove "Concierge" flow where users can share links or photos of items directly via WhatsApp.
3.  **Scan & Ship**: A digital documentation flow for uploading invoices or delivery details for automated processing.

## 🛠 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Vanilla CSS (CSS Modules) + Glassmorphism
- **Branding**: Custom SVG-based Premium Design System

## 📂 Project Structure
- `src/app`: Application routes (Home and Dashboard)
- `src/components`: Reusable UI components (Logo, etc.)
- `src/app/dashboard/shipping_data.json`: Master data for item weights and categories.

## 🏃 Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the landing page.
Navigate to [/dashboard](http://localhost:3000/dashboard) to try the shipment options.
